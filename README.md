# 🎨 Aarthy Venkat – Portfolio Website

A modern personal portfolio website showcasing my creative design works, including automotive posters, branding concepts, and visual experiments.

---

## 📌 Overview

This project is a multi-page portfolio website built using **HTML, CSS, and basic web design principles**. It includes:

- Home / Portfolio page
- Design showcase page
- Interactive UI elements
- Responsive card-based layout
- Contact / Ask Me Anything section

---

## 🖼️ Features

- 🎯 Clean modern UI design
- 🖥️ Responsive grid layout for works
- 🏎️ Automotive-themed design posters
- 🏠 Home navigation icon linking pages
- ✉️ Email contact link (mailto)
- 💬 Ask Me Anything form section
